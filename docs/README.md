# Padh.ai Documentation

This directory contains the documentation for the Padh.ai project.

## Project Overview

Padh.ai is a platform for students and faculty to manage lab resources, participate in hackathons, and access educational materials.

## Architecture

The project follows a microservices architecture, with the following components:

-   **Frontend:** A React application that provides the user interface.
-   **API Gateway:** An Express.js application that acts as the single entry point for all API requests.
-   **Auth Service:** A Node.js service for handling user authentication.
-   **Notification Service:** A Go service for handling notifications.
-   **PDF Hosting Service:** A Go service for storing and serving PDF files.// needed for displaying playbooks , might remove and shift to better alternative soon
-   **Database:** A PostgreSQL database for storing all the application data.
